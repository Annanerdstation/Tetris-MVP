### Prerequisites

- Node.js (version 16 or higher)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/annagorozia/tetris-mvp.git
   cd tetris-mvp
   ```

   ```bash
   npm install
   ```
